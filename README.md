# Hey, soy IsaSan 👋

Soy full stack developer, aunque "stack" a veces es solo una forma elegante de decir que rompo cosas en el frontend *y* en el backend por igual.

Ahora mismo estoy construyendo proyectos para asentar conceptos: aplicaciones con Java y Angular por un lado, y por el otro aprendiendo a automatizarlo todo con GitHub Actions. La combinación perfecta para que algo falle en tres sitios a la vez.

---

## Un poco sobre mí

- 🔧 En el día a día: Java, JavaScript, Angular, Docker y Oracle. Y burocracia. Sobre todo burocracia
- ☕ Explorando Java y Angular para ampliar el stack
- 🤖 Aprendiendo GitHub Actions: automatizar cosas aunque nadie me lo haya pedido
- 🐛 Mi proceso de debug favorito: añadir `console.log` hasta que algo tenga sentido
- 🍫 Unidad de medida del progreso: Reese's por bug resuelto
